### 우분투용용의 avconv 이용


``` bash 형식
avconv  -i [영상파일] -vn  [소리파일]
```

``` bash 예시
avconv  -i Into.the.Wild.2007.1080p.BluRay.x264.YIFY.mp4 -vn Into.the.Wild.2007.1080p.BluRay.x264.YIFY.mp3
```
